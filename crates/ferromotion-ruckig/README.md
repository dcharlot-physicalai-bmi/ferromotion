# ferromotion-ruckig

[![crates.io](https://img.shields.io/crates/v/ferromotion-ruckig.svg)](https://crates.io/crates/ferromotion-ruckig)
[![docs.rs](https://img.shields.io/docsrs/ferromotion-ruckig)](https://docs.rs/ferromotion-ruckig)

Jerk-limited online trajectory generation in pure Rust (native + `wasm32`) — a clean-room port of the
[Ruckig](https://github.com/pantor/ruckig) community-edition method: time-optimal, jerk-constrained
motion profiles for real-time control.

Part of [Ferromotion](https://github.com/dcharlot-physicalai-bmi/ferromotion). Dual-licensed MIT OR Apache-2.0.
